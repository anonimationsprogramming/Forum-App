# Version 1

## What is this app?

- This is the first version of a forum engine that supports registry, login, thread creation, and replying.  
- This version is a very minimalist prototype but shows different aspects of architecture, modularization, and a few other C++ programming skills I know.  

## How to compile and run?

- Linux

1. Download: ``` sudo apt install build-essential ``` -> This bundle contains make (to build C or C++ programs with makefiles and also contains gcc and g++ which are the compiler-linkers for C and C++).  
2. Open the terminal in the v1 folder, next to the makefile: ``` cd /path/to/folder/v1 ```  
3. Run: ``` make ```  
4. A file named ``` forum-app ``` will be created inside the v1 directory along with an obj directory. Still inside v1, run in terminal: ``` ./forum-app ```  
5. It will run a CLI app asking for different data and confirmations.
